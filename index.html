float-labels.js

This zero-dependency plugin applies the float label pattern to a form.

The float label pattern floats the inline label up above the input after the user focuses on the form field or enters a value.

For production, use the files from the dist/ folder.

Default Usage

Using vanilla javascript

<link href="css/float-labels.css" rel="stylesheet">
<script src="js/float-labels.min.js"></script>

<script>
	var floatlabels = new FloatLabels( document.querySelectorAll( 'form' ));
</script>

Using jQuery

<link href="css/float-labels.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.1/jquery.slim.min.js"></script>
<script src="js/float-labels.min.js"></script>

<script>
	$( 'form' ).floatlabels();
</script>

Options

Here are the default options

{
	customEvent: null,
	customLabel: null,
	exclude    : '.no-label',
	inputRegex : /email|number|password|search|tel|text|url/,
	prioritize : 'label',
}

Build

float-labels.js uses gulp to build from src.

$ npm install
$ gulp

The compiled files will be saved in the dist/ folder.
